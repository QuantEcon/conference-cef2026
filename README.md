# QuantEcon · CEF 2026 pre-conference

Materials for QuantEcon's session at the **CEF 2026 pre-conference** (Venice,
28 June 2026) — the proposal for a community-owned, cross-toolkit **reference
library for computational economics**.

## Take part

Projects interested in participating can **register** by opening an issue from
the registration template (*Issues → New issue*). Open `registration` issues
form the live list of participating projects — each project keeps credit and
editorial control of its own content.

The proposal: each participating project contributes a **shared baseline model**
(coordinated across its group) and a **hero tutorial** (its author's-choice
showcase), authored and maintained by the project, and hosted together as the
field's first cross-toolkit reference.

## In this repo

- **Registration** — a project registration issue form (`.github/ISSUE_TEMPLATE/`).
- **Slides** — the reveal.js talk (Quarto source under `slides/`), published via
  GitHub Pages.
